# IoT Medication Reminder App

An IoT medication adherence assistant with an Expo mobile workflow and a Vite web landing page for the product experience.

## Live Demo

Web application: https://iot-medication-reminder-app.vercel.app

## Repository

GitHub: https://github.com/anushka9480-alt/IOT_PROJECT

## Project Structure

- `App.tsx` and `src/` contain the Expo mobile medication assistant.
- `IoT Medication Reminder App/` contains the Vite React frontend deployed on Vercel.
- `backend/` contains the API and Prisma backend scaffolding.

## Run Locally

Install root dependencies and run the mobile app:

```bash
npm install
npm run start
```

Run the web frontend:

```bash
cd "IoT Medication Reminder App"
npm install
npm run dev
```
