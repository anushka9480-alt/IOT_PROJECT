
# IoT Medication Reminder App

Web landing page for the IoT Medication Reminder App, featuring a responsive grid layout, parallax scrolling, reveal animations, and a loading experience.

## Live Demo

Production URL: https://iot-medication-reminder-app.vercel.app

APK link: https://github.com/anushka9480-alt/IOT_PROJECT/releases/latest/download/medicine-adherence-assistant.apk

## Design Source

The original project is available at https://www.figma.com/design/YMB272xtcqmg533R1Aoimb/IoT-Medication-Reminder-App.

## Running the code

Run `npm install` to install the dependencies.

Run `npm run dev` to start the development server.
